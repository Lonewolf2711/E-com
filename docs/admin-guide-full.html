Admin Complete Guide

Everything you need to manage the Spare Parts Portal — from adding parts and handling enquiries to configuring email, WhatsApp, and AI quotation generation.

🔐 Logging Into the Admin Panel

Go to: yourwebsite.com/admin
1

Open the Admin Login Page

In your browser, go to your website address followed by /admin. Example: http://localhost/E-com/admin

2

Enter Your Credentials

Enter your Email Address and Password. These are set up during installation. If you've forgotten your password, contact your web developer.

3

You're In!

After logging in, you'll land on the Dashboard — your main control centre.

💡 Tip: Always log out when you're done, especially on shared computers. Use the logout option in the top-right of the admin panel.

⚙️ Managing Parts (Products)

Admin → Products

This is where you manage your entire spare parts catalogue — add, edit, and delete parts.

Adding a New Part

1

Click "Add New Part" or "+ Add Product"

From the dashboard (quick action) or Products page, click the button to open the add product form.

2

Fill In: Product Information

  • Product Name ⭐ Required — the full name of the part
  • Part Code ⭐ Required — unique customer-facing code (e.g. SP-BEAR-001). This is what customers search for and mention in enquiries.
  • Machine Name — main machine this part belongs to
  • Machine Model — specific model number (e.g. G6300)
  • Compatible Machines — comma-separated list of all machines this part works with
  • Short Description — one-line summary (max 255 characters)
  • Full Description — detailed technical description
3

Fill In: Pricing & Availability

  • Internal Price (₹) — your cost/reference price. Not shown to customers.
  • Sale/Quoted Price (₹) — used internally for quote calculations. Still not public.
  • Stock Quantity — number of units available
  • Internal SKU / Reference Code — your internal reference (different from the public Part Code)
4

Upload Images

  • Main Image — the primary photo shown on cards and the detail page
  • Gallery Images — additional photos (multiple files at once)
5

Add Attributes (Optional but Recommended)

Click "+ Add" to create attribute pairs like:

  • Name: Inner Diameter — Value: 25mm
  • Name: Material — Value: Chrome Steel
  • Name: Standard — Value: ISO 6206
6

SEO Information (Recommended)

  • Meta Title — what appears in Google search results (max 70 characters). Include part code and machine name.
  • Meta Description — the description shown below the title in Google (max 160 characters)
7

Sidebar Settings

  • Category ⭐ Required — select the category this part belongs to
  • Status — Active = visible on website. Inactive = hidden.
  • Featured Product — tick this to show the part in the "Featured" section on the homepage
  • Available on Request — tick this if the part isn't in stock but can be sourced. Shows "On Request" badge.
8

Save the Part

Click "Create Product" (or "Update Product" if editing). The part will appear in the catalogue immediately.

⚠️ Part Code Tips: Use a consistent format like [TYPE]-[CATEGORY]-[NUMBER] (e.g. SP-BEAR-001 for spare part, bearing, number 1). This helps customers find parts by code and helps with Google search.

Editing an Existing Part

1

Go to Admin → Products

You'll see a list of all parts.

2

Click "Edit" on the Part

The edit form opens — identical to the add form, but pre-filled with existing data.

3

Make Changes and Save

Update what you need and click "Update Product."

Deleting a Part

On the Products list, find the part and click the Delete button. You'll be asked to confirm. Once deleted, the part is removed from the catalogue.

⚠️ Warning: Deleting a part is permanent. If a product is linked to past enquiries, you may want to set it to Inactive instead of deleting it.

🏷️ Managing Categories

Admin → Categories

Categories organise your parts. Examples: Bearings, Gears, Drive Belts, Sensors, Motors.

1

Add a New Category

Click "+ Add Category." Enter a name, upload an icon/image, and optionally a parent category if you want sub-categories.

2

Category Fields

  • Name — e.g. "Bearings"
  • Slug — auto-generated URL-friendly name (e.g. bearings)
  • Image — icon that appears on the homepage category carousel
  • Parent Category — for sub-categories
  • Status — Active or Inactive
3

Remove a Category

Only do this if the category has no products. Products must be moved to another category first.

📦 Inventory / Stock Management

Admin → Inventory

The Inventory page lets you quickly update stock levels for all parts without going into each product's edit form.

Understanding Stock Colours

Badge ColourMeaning
Green NumberHealthy stock — plenty available
Yellow NumberLow stock — getting close to the threshold
Red NumberOut of stock or at zero — urgent restock needed

Quick Stock Adjustment

1

Find the Part

Use the search bar to find by part name or SKU code.

2

Choose the Operation

  • + (Add) — add units to current stock (e.g. received new shipment)
  • – (Subtract) — remove units from current stock
  • = (Set) — set the stock to an exact number
3

Enter the Quantity and Click "Update"

The stock level updates immediately.

Low Stock Alert Page

Click "Low Stock Alerts" button to see only the parts that are running low. This is also shown on the dashboard.

Right Panel — Email & AI Tools

The right panel has two powerful tools for responding to enquiries:

  1. Load from Template — pick a pre-written email template and load it instantly
  2. AI Quote Generator — let the AI write a professional quotation email for you

Quick Contact Buttons

Below the email tools, there are one-click buttons to email the customer or open WhatsApp with their number pre-filled.

🤖 AI Quote Generator

The AI can automatically write a professional quotation email based on the enquiry details. This saves time and ensures a professional tone.

ℹ️ Requirement: To use the AI generator, you need a Groq API key configured in Settings. See the AI Setup section for how to get one (it's free).
1

Open Any Enquiry

Go to Admin → Enquiries → click on any enquiry to open its detail page.

2

Choose a Tone

In the AI section on the right, select the email tone:

  • Formal & Professional — for corporate clients
  • Friendly & Helpful — for regular customers
  • Follow-up / Reminder — when following up on a pending quote
  • Urgent / Time-sensitive — for rush requirements
3

Add Notes (Optional)

Type any specific information in the "Additional Notes" box. For example: "Price valid for 7 days. Delivery 3–5 working days. Minimum order 10 units."

4

Click "Generate with Groq AI"

The system sends the enquiry details to the AI. In a few seconds, a complete email — with Subject and Body — will appear.

5

Review and Edit

Read through the generated email. Edit the Subject or Body in the text boxes if anything needs changing. The AI always uses the customer's name and part details automatically.

6

Send or Copy

  • "Copy" — copies the email text to your clipboard to paste into your own email client
  • "Send to Customer" — sends the email directly to the customer via the system (requires SMTP configured)

✉️ Email Templates

Admin → Email Templates

Email Templates let you create reusable email responses. Instead of writing the same message every time, you save it as a template and load it with one click.

Creating a Template

1

Click "+ Add Template"

Go to Admin → Email Templates and click the add button.

2

Fill In the Template

  • Name — internal name to identify it (e.g. "Standard Quotation Reply")
  • Type — choose "Enquiry" for templates used in enquiry responses
  • Subject — the email subject line
  • Body — the full email content. Use placeholders:

Placeholders You Can Use

PlaceholderWhat It Inserts
{{customer_name}}Customer's full name
{{enquiry_number}}The enquiry reference number (e.g. ENQ-0001)
{{customer_email}}Customer's email address
{{customer_phone}}Customer's phone number
{{customer_company}}Customer's company name

Using a Template on an Enquiry

On the enquiry detail page (right panel), you'll see a "Load from Template" section. Select your template from the dropdown and click "Load" — the email is pre-filled with all placeholders replaced.

🖼️ Homepage Banners

Admin → Banners

Banners control the large hero slider at the top of your homepage. You can add image slides or video slides (YouTube or MP4).

Adding a New Banner Slide

1

Click "+ Add Banner"

Opens the banner creation form.

2

Banner Content

  • Title — the large heading text on the slide (e.g. "Precision Parts. Fast Quotations.")
  • Subtitle — small text that appears above the title (e.g. "Industrial Spare Parts")
  • Description — a short paragraph below the title
3

Call to Action Button

  • Button Text — what the button says (e.g. "Browse Parts")
  • Button Link — where it goes when clicked (e.g. /shop or /category/bearings)
4

Choose Media Type: Image or Video

  • Image Slide — upload a photo (recommended size: 1920×600px)
  • Video Slide — paste a YouTube link (e.g. https://youtu.be/XXXXXXXXXXX) or upload an MP4 file (max 50MB)
5

Settings

  • Position — choose "Hero Slider" (the main scrolling slideshow)
  • Sort Order — lower number = appears first in the slideshow
  • Status — Active = shown on website. Inactive = hidden.
6

Save and Check Live

Click "Create Banner" and then click "View Live Website" on the dashboard to see it.

💡 Slideshow: If you add multiple banners with Position = "Hero Slider," they automatically become a slideshow that rotates every 5.5 seconds with smooth fade transitions.
ℹ️ Video Tips: YouTube videos play muted and looped as backgrounds. If using an MP4, keep it under 50MB. The slide always has a dark overlay so your text stays readable.

🎟️ Discount Coupons

Admin → Coupons

Create discount codes that customers can use at checkout.

Creating a Coupon

1

Click "+ Add Coupon"

A popup form appears.

2

Fill In the Coupon Details

FieldMeaning & Example
Coupon CodeWhat customers type at checkout (e.g. SUMMER20). Always uppercase.
Discount TypePercentage (%) — takes a % off the total. Fixed Amount — deducts a flat ₹ amount.
Discount ValueThe amount. E.g. 20 for 20% off. Or 500 for ₹500 off.
Max DiscountFor percentage type: caps the maximum discount (e.g. max ₹1000 off). Leave 0 for no cap.
Min Order AmountMinimum cart value before coupon can be used. Leave 0 for no minimum.
Max UsesHow many times this code can be used total. Leave 0 for unlimited.
Expiry DateWhen the coupon expires. After this date, the code no longer works.
3

Click "Create Coupon"

The coupon is now active and customers can use it at checkout.

Coupon Performance

Click "Performance" button on the Coupons page to see which coupons have been used most and the total discount given.

Deactivating or Deleting a Coupon

On the coupons list, click the red trash icon 🗑️ to delete a coupon entirely. Once deleted, the code no longer works for customers.

📈 Reports

Admin → Reports

Reports give you data about your business performance. There are three report types.

Sales Report

Shows revenue and orders over time. Select a period (Last 7 Days, 30, 90, or 365 days). You'll see:

  • Total Revenue — total sales income in the period
  • Total Orders — how many orders were placed
  • Average Order Value — revenue ÷ orders
  • Daily Revenue Chart — line graph showing revenue day by day
  • Payment Method Breakdown — which payment methods customers used (Cash on Delivery, Razorpay, etc.)

Products Report

Shows which parts are selling/being enquired about most. Useful for deciding what to keep in stock.

Customers Report

Shows your top customers by spend and order frequency.

🔍 SEO Management

Admin → SEO

SEO (Search Engine Optimisation) helps your parts appear in Google searches — especially when customers search by part code or machine name.

Product SEO

1

Select a Part

Choose from the "Select Part" dropdown.

2

Fill In SEO Fields

  • Meta Title — what Google shows as the link title. Best format: [Part Code] [Part Name] for [Machine] | [Store Name]
  • Search Keywords — include part code, machine models, and part type. Example: SP-BEAR-001, deep groove bearing, Sulzer G6300, loom bearing
  • Meta Description — the short summary Google shows under the title. Max 160 characters. Should mention the part code and what it's compatible with.
  • OG Image URL — the image shown when shared on social media (usually the product image URL)
3

Save

Click "Save Product SEO."

💡 SEO Formula for Spare Parts:
Meta Title: SP-BEAR-001 Deep Groove Ball Bearing for Sulzer G6300 | YourStore
Meta Description: Buy SP-BEAR-001 bearing compatible with Sulzer G6300, G6200. Enquire for pricing and fast dispatch.

Category SEO

Works the same as Product SEO but for category pages. Select the category tab, pick a category, fill in the fields.

Global Settings (Google Analytics & Search Console)

FieldWhat To Do
Google Search Console VerificationPaste the content from the Google verification meta tag (from Google Search Console → Verify site → HTML tag method)
Google Analytics Measurement IDYour GA4 ID (starts with G-). Find it in Google Analytics → Admin → Data Streams
Default Meta Title TemplateTemplate used for parts that don't have custom SEO. Default: {product_code} — {name} for {machine_name}

Email Configuration (SMTP)

SMTP settings are required for the system to send emails (enquiry notifications, quotations). Without this, emails may not be delivered.

FieldValue
SMTP Hostsmtp.gmail.com for Gmail, smtp.office365.com for Outlook
SMTP Port587 for TLS (recommended), 465 for SSL
EncryptionTLS (recommended for most providers)
SMTP UsernameYour full email address (e.g. yourname@gmail.com)
SMTP PasswordFor Gmail: Use an App Password — not your regular Gmail password. See steps below.
From EmailThe email address that appears as sender (e.g. noreply@yourcompany.com)
From NameThe name that appears as sender (e.g. "Spare Parts Store")

Setting Up Gmail App Password (Step by Step)

1

Enable 2-Step Verification on your Google Account

Go to myaccount.google.com → Security → 2-Step Verification → Turn it on.

2

Create an App Password

Go to myaccount.google.com/apppasswords → Select app: "Mail" → Select device: "Other" → Name it "Spare Parts Store" → Click Generate.

3

Copy the 16-character Password

Google shows a 4-word code like xxxx xxxx xxxx xxxx. Copy it.

4

Paste into SMTP Password Field

In Settings → SMTP Password, click the eye icon to show the field, and paste the app password.

5

Save Settings

Click "Save Settings" at the bottom.

AI Quotation Generator Setup (Groq)

Groq is a free AI service that powers the quote generator. Follow these steps to get your free API key:

1

Go to console.groq.com

Open your browser and visit https://console.groq.com. Create a free account.

2

Create an API Key

In the Groq dashboard, go to API Keys → Create New Key → give it a name like "Spare Parts Store."

3

Copy the API Key

It starts with gsk_.... Copy it immediately — Groq only shows it once!

4

Paste in Settings

In Admin → Settings → AI Quotation Generator (Groq) section → paste the key in the Groq API Key field → click eye icon to reveal → click Save Settings.

✅ Free Tier: Groq's free plan is extremely generous — more than enough for generating quotation emails daily.

Saving Settings

After making any changes in Settings, scroll to the bottom and click "Save Settings." All settings are saved together in one click.
